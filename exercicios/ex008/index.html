Principais formatações


Negrito / Destaque

Nesta frase temos um termo em destaque usando a tag <strong> e </strong>.


Itálico / Ênfase

Nesta frase temos um termo em destaque, usando a tag <em> e </em>.


Texto marcado

Podemos também marcar um texto usando a tag <mark> e </mark>

E no outro parágrafo marcado no final.


Textos grandes e pequenos

Estamos criando nesse parágrafo um texto grande e um texto pequeno.


Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lida, mas desconsiderado.


Texto inserido

Podemos marcar um texto como inserido para dar ênfasa e indicar que ele foi inserido depois.


Texto sobrescrito

Podemos usar para inserir coisas do tipo x20+3


Texto subescrito

Podemos usar para inserir coisas do yipo h20
